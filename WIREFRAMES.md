7. QR code hidden (only secret shown if needed)

## Design System

### Colors
- **Primary:** Blue (#2563eb) - Buttons, links, primary actions
- **Secondary:** Purple (#9333ea) - MFA, special features
- **Success:** Green (#10b981) - Success states, complete actions
- **Danger:** Red (#ef4444) - Delete, logout, errors
- **Neutral:** Slate (#64748b) - Text, borders, backgrounds

### Typography
- **Headings:** Bold, 2xl (24px)
- **Body:** Regular, base (16px)
- **Small:** Regular, sm (14px)
- **Tiny:** Regular, xs (12px)

### Spacing
- **Card Padding:** 8 (32px)
- **Form Spacing:** 4 (16px)
- **Button Padding:** 2.5px vertical, 4px horizontal (10px 16px)

### Components
- **Cards:** White background, rounded-xl, shadow-lg, border
- **Buttons:** Rounded-lg, hover states, disabled states
- **Inputs:** Border, rounded-lg, focus ring
- **Badges:** Rounded, colored background, small text
- **Icons:** 4x4 (16px) inline, 6x6 (24px) standalone
